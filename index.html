As you start out with backbonejs, you'll likely feel uncertain as to what classes to make first. Below is the most sensical way to do things.

Order of Operation

  • Model Class
  • View Class

  • Model Instantiation
    Requires:
    • Model Class

  • View Instatiation
    Requires:
    • Model Class
    • View Class
    • Model Instantiation

  • Collection Class
    • Requires:
    • Model
